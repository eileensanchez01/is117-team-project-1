    AFS Files and Folders (Terminal)

    Public Comping Labs: You can access the files on your AFS account using MobaXterm on the PCs in the Public Computing Labs. Software to Access AFS (Note: VPN is NOT required to access AFS from off-campus)

    Windows: MobaXterm provides command line and GUI access to your AFS account on a Linux host from a Windows host as well as the ability to transfer files between your computer and your AFS account using SFTP or SCP. MobaXterm supports other features.

    Go to the software download page to get the latest version of MobaXterm. For information on how to connect using MobaXterm go to http://ist.njit.edu/how-connect-afs-mobaxterm/. Mac OS X: If you are using Mac OS X, click here for SSH installation instructions or go to

    https://wiki.linux.njit.edu/index.php/Installing_an_AFS_Client_on_MacOSX (Mac 10.13 High Sierra). Ubuntu (Linux): For instructions about accessing AFS on Ubuntu please go to https://wiki.linux.njit.edu/index.php/Installing_an_AFS_Client_on_Ubuntu.
